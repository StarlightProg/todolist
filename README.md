Сайт для создания собственного To-Do-List, с возможностью добавления задачам тегов, сортировки по тегам

Установка:
1) Создать .env
2) Открыть терминал
3) php artisan key:generate
4) php artisan storage:link
5) php artisan migrate
6) php artisan serve
7) Открыть другой терминал
8) npm install
9) npm run dev
